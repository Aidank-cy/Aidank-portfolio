# Visual Rules

- Match an Apple / Linear / Vercel style direction.
- Prefer black, white, gray, and restrained neutral tones as the base palette.
- Support dark mode.
- Use one accent color only for CTA and key links.
- Manage color tokens through shared CSS variables or a single global theme layer, not scattered one-off values.
- Use large editorial typography and generous white space.
- Section spacing should be generous; target at least `120px` vertical padding between major sections on larger layouts.
- Keep content width constrained to roughly `1200px` and centered.
- Avoid noisy gradients and over-decorated UI.
- Typography should have clear identity:
  - choose distinctive `next/font` families
  - do not use Inter, Arial, or Roboto as the main design voice
- Motion should be subtle and support hierarchy rather than decoration.
