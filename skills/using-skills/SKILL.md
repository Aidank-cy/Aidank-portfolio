# Using Skills

Use this router before starting larger tasks in this repository.

## Choose The Local Skill

- Use `skills/static-export-guard/SKILL.md` for route, config, workflow, or build/export changes.
- Use `skills/github-portfolio-data/SKILL.md` for project entries, profile copy, GitHub data mapping, or README rendering work.
- Use `skills/ui-polish/SKILL.md` for layout, motion, typography, or component-level visual changes.
- Use `skills/versioning-and-changelog/SKILL.md` after meaningful repo changes that should be recorded in `CHANGELOG.md`.

## Shared Rules

- Start with `AGENTS.md`.
- Read only the `.project-rules/` files relevant to the task.
- Keep changes source-first; never patch `.next/` or `out/`.
